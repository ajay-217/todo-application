# To-Do List Project

## Project Structure


## Frontend View Layer – UI Developer

This layer is responsible for:
- Designing the user interface (UI)
- Creating HTML templates for task listing and editing
- Ensuring responsive and user-friendly layouts using Bootstrap

### Files handled:
- `templates/index.html` – Main To-Do list UI
- `templates/edit_task.html` – Edit task UI

> Backend logic and database operations are handled by other team members.
